<Section icon={Shield} title="PC System Requirements and Performance Targets">

Gears of War: E-Day is built from the ground up using Unreal Engine 5, delivering a massive visual leap for the franchise. The game utilizes advanced rendering techniques, micro-destruction systems, and real-time ray tracing. To run the game smoothly on PC, players must meet specific hardware baselines.

The following tables outline both the minimum and recommended system configurations required for the PC version, as listed on the official [Steam Store Page](https://store.steampowered.com/app/3010850/Gears_of_War_EDay/).

### Minimum PC Requirements
| Hardware Component | Minimum Specification |
| :--- | :--- |
| Operating System | Windows 10 64-bit (Version 22H 19045.7291) |
| Processor | AMD Ryzen 5 2600X / Intel Core i7-6850K / Intel Core i5-10400 |
| Memory | 12 GB RAM |
| Graphics Card | NVIDIA RTX 5050 / NVIDIA RTX 2060 / AMD RX 6600 / Intel Arc A580 |
| DirectX | Version 12 |
| Storage Space | 130 GB available space (Solid State Drive required) |

### Recommended PC Requirements
| Hardware Component | Recommended Specification |
| :--- | :--- |
| Operating System | Windows 11 64-bit (Version 25H2 or newer) |
| Processor | AMD Ryzen 5 5600 / Intel Core i5-11600K |
| Memory | 16 GB RAM |
| Graphics Card | NVIDIA RTX 5060 / NVIDIA RTX 3060 Ti / AMD RX 6700 XT / Intel Arc B580 |
| DirectX | Version 12 |
| Storage Space | 130 GB available space (Solid State Drive required) |

Because of the high-fidelity environmental destruction, rapid asset streaming, and complex lighting, a Solid State Drive (SSD) is a mandatory requirement for both minimum and recommended setups. Mechanical hard drives (HDDs) are not supported and will experience severe hitching and loading issues.

The game also supports modern display technologies including HDR10, Variable Refresh Rate (VRR), and Dolby Atmos spatial audio.
</Callout>
